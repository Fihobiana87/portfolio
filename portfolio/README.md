# Portfolio - Fihobiana RAZAFINDRAIBE

Portfolio professionnel statique de Fihobiana RAZAFINDRAIBE, jeune entrepreneur passionne par le marketing digital, le commerce et les nouvelles technologies.

## Structure

- `index.html` : accueil, hero, competences, projet vedette, timeline
- `about.html` : profil, parcours, objectifs, realisations
- `projects.html` : presentation detaillee du projet E-Kata avec galerie et lightbox
- `contact.html` : formulaire de contact valide en JavaScript
- `assets/css/style.css` : direction artistique, responsive, animations
- `assets/js/main.js` : interactions, reveal au scroll, slider, lightbox, formulaire

## Palette

La direction visuelle reprend les couleurs dominantes observees dans les fichiers fournis : encre noire, bleu nuit, ivoire clair, gris premium et accent cuivre/orange.

## Deploiement

Le projet est statique et directement deployable sur Netlify, Vercel, GitHub Pages ou tout serveur web classique. Ouvrir `index.html` suffit pour un apercu local.
